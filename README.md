# mule-oauth2-googledrive
This solution is used to demonstrate use of JWT to retrieve files from google drive.

### Configurations:
replace the contents of "mule-googledrive-serviceacct.json" file under src/main/resources with your google drive service account keys
To test, run the application and navigate to http://localhost:8081

###
More details can be found at the blog post: https://rumanblogs.com/authenticating-with-google-drive-in-mule-part-2-jwt/
